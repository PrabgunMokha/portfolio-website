# Portfolio of Prabgun Mokha

This is a static portfolio site ready to deploy on Vercel.

## Files

- `index.html` - main website
- `vercel.json` - Vercel rewrite config
- `package.json` - project metadata and helper scripts

## Run locally

Open `index.html` directly in your browser, or run:

```bash
npm run dev
```

## Deploy to Vercel

From this folder, run:

```bash
vercel
```

For a production deployment:

```bash
vercel --prod
```
